## 🚀 Featured Projects

### 📊 [Backend-Ledger](https://github.com/Sahilyadhuvansi/Backend-Ledger) — *AI-Native Financial Suite*

> A professional ledger system evolved into an AI-powered financial platform. Deployed live at [mini-bank-beta.vercel.app](https://mini-bank-beta.vercel.app)

**✨ AI & Production Features:**

- 🤖 **AI Financial Advisor** — Groq-powered advisor providing localized insights (INR) based on transaction history.
- 📄 **AI Receipt Scanner** — OCR-driven document processing to automate expense logging via `receipt-scanner.service.js`.
- 🛡️ **Intelligent Fraud Detection** — AI monitoring system that identifies suspicious transaction patterns in real-time.
- 🏦 **Double-entry ledger model** — Immutable debit/credit entries with ACID-compliant multi-document sessions.
- ⚡ **Socket.io live updates** — Real-time balance updates and transfer notifications using a centralized Event Emitter.
- 🔒 **Security hardened** — JWT with HTTP-only cookies, token blacklisting (auto-expiring TTL), and strict rate limiting.

**Tech:** `Node.js` `Express` `MongoDB` `Groq AI` `Socket.io` `JWT` `React` `Tailwind CSS`
